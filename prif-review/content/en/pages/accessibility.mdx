This accessibility statement applies to PRIF Review (<a href="https://review.prif.org/en/">review.prif.org</a>), a website run by [Peace Research Institute Frankfurt](https://www.prif.org). We want as many people as possible to be able to use this website. For example, that means you should be able to:

- zoom in up to 300% without the text spilling off the screen
- navigate most of the website using just a keyboard
- navigate most of the website using speech recognition software
- listen to most of the website using a screen reader

We’ve also made the website text as simple as possible to understand. [AbilityNet](https://www.abilitynet.org.uk/) has advice on making your device easier to use if you have a disability.

## How accessible this website is

To our knowledge, this website is fully accessible. This was established in internal tests carried out by our development team.

## Feedback and contact information

If you need information on this website in a different format like accessible PDF, large print, easy read, audio recording or braille, pleace contact us at [info@prif.org](mailto:info@prif.org). We will consider your request and respond within five working days.

## Reporting accessibility problems with this website

We’re always looking to improve the accessibility of this website. If you find any problems not listed on this page or think we’re not meeting accessibility requirements, please contact us at [info@prif.org](mailto:info@prif.org).

## Enforcement procedure

The _Hessisches Ministerium für Soziales und Integration_ is responsible for enforcing the _Hessische Verordnung über barrierefreie Informationstechnik_ which establishes the accessibility requirements for public sector websites like this one. ## Technical information about this website's accessibility

This website is fully compliant with the [Web Content Accessibility Guidelines v2.1](https://www.w3.org/TR/WCAG21/) AA standard. Its compliance status was established in internal tests.
