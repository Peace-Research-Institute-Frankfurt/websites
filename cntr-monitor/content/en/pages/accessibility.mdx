This accessibility statement applies to CNTR Monitor ([monitor.cntrarmscontrol.org](https://monitor.cntrarmscontrol.org)), a website run by the [Cluster for Natural and Technical Science Arms Control Research](https://www.cntrarmscontrol.org/).

As a public-sector institution, we are comitted to enable as many people as possible to use this website. For example, that means you should be able to:

- zoom in up to 300% without the text spilling off the screen
- navigate the website using just a keyboard
- navigate the website using speech recognition software
- listen to most of the website using a screen reader

We’ve also made the website text as simple as possible to understand.

## How accessible this website is

To our knowledge, this website is fully accessible.

## Technical information about this website's accessibility

Our accessibility requirements originate from _Hessische Verordnung über barrierefreie Informationstechnik_ [HVBIT] [Hessian Directive on Accessible Information Technology] § 3 Section 1-4, implementing EU Directive 2016/2102. They are equivalent to the [Web Content Accessibility Guidelines 2.1](https://www.w3.org/TR/WCAG21/) AA Standard.

This website is fully compliant with HVBIT. Its compliance status was established in internal tests.
